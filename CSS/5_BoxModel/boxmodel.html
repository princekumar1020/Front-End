Box Model:

Padding : Space between Content and Border

Border : the visible line outside padding

Margin : outside border

This short Hand property same for all border,padding,margin.

padding: [top] [right] [bottom] [left];

margin: 10px 20px 30px 40px;
10px → Top
20px → Right
30px → Bottom
40px → Left

padding: [top] [right and left] [bottom];

margin: 10px 20px 40px;
10px → Top
20px → Right And Left
40px → Bottom

padding: [vertical] [horizontal];

margin: 10px 20px;
top and bottom : 10px ;
left and right : 20px ;

padding: [vertical] [Auto];

margin: 10px auto; : for taking box in center margin: auto doesn’t work on inline elements because they don't accept width, and without a fixed width, auto margins can't calculate space for centering.
top and bottom : 10px ;
left and right : 20px ;

padding: [All sides];

margin: 10px;
All side 10 px
